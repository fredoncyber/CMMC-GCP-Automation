---

## 🧭 Overview
**CMMC-GCP-Automation** provides a semi-automated approach to achieving **CMMC Level 1 and Level 2 readiness** within Google Cloud.  
It collects and maps evidence from IAM, VPC, Cloud Logging, and Security Command Center, while generating dashboards, SSP/POA&M templates, and compliance reports.

---

## ⚙️ Features
✅ Hybrid control-to-evidence mapping for CMMC L1 & L2  
✅ Automated evidence collection via GCP SDKs & CLI  
✅ JSON + CSV output for GRC integration  
✅ GitHub Actions CI/CD for continuous monitoring  
✅ SSP & POA&M templates for assessor documentation  
✅ Modular design to expand across all 14 CMMC domains
